# Pokedexcli

A command-line Pokedex built in Go that uses the [PokeAPI](https://pokeapi.co/).

## Usage

```sh
go run .
```

## Commands

| Command   | Description                          |
|-----------|--------------------------------------|
| `map`     | Retrieve locations from the map      |
| `bmap`    | Retrieve previous locations          |
| `explore` | List pokemon in a location           |
| `catch`   | Try to catch a pokemon               |
| `inspect` | View a caught pokemon's stats        |
| `pokedex` | See all caught pokemon               |
| `help`    | Display help                         |
| `exit`    | Exit the Pokedex                     |
